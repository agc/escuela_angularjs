        // un iteratee que consume chunks Strings y produce un Int

        El interface Iteratee Iteratee[E,A] tiene dos parámetros takes E, que representa el tipo de la Entrada que acepta y A el tipo del resultado que se calcula

        Los iteratee se crean independientes de cualquier fuente de datos. La fuente de datos es responsable de alimentar un iteratee con el primer chunk, el cuál produce otro iteratee que será alimentado con el siguiente chunk, y así sucesivamente
